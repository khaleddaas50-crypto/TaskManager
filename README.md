# TaskManager

## Project Description
TaskManager is a simple Java application that demonstrates Object-Oriented Programming (OOP) concepts.
The project allows users to create projects, add tasks, manage deadlines, priorities, and track task completion.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- JUnit 5
- Git & GitHub

## How to Run Main
1. Open the project in Eclipse
2. Navigate to:
   src/com/khaled/taskmanager/Main.java
3. Right click on Main.java
4. Select: Run As → Java Application

## How to Run JUnit Tests
1. Open the `test` folder
2. Right click on any test class (TaskTest or ProjectTest)
3. Select: Run As → JUnit Test
